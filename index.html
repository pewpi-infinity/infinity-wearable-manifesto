🔬 Simulation Modules (A-D + Core)

Module A

maxwell_field_sim.py

Unified electromagnetic analysis. Gauss, Ampere, Faraday laws applied to ion systems. EM spectrum mapping for wearable bands. Field energy density comparison.

Module B

thermal_safety_sim.py

IEC 60601-1 safety limits. Component heat generation with duty cycles. Thermal resistance stack model. Graphene heat spreader analysis. Skin temperature prediction.

Module C

memory_map_sim.py

Distributed RAM across 24 mesh nodes. 24-bit address space. Data flow pipeline: sense → filter → compress → route → infer → report. Energy-memory equivalence analysis.

Module D

power_density_sim.py

Textile power grid with IR drop simulation. Poisson relaxation solver. Stretch-zone resistance mapping. Thermal feedback from I²R hotspots. Grid optimization recommendations.

Core

ion_flux_sim.py

Nernst-Planck ion transport. Na+, K+, Ca2+, Cl- membrane fluxes. Diffusion (Fick) + migration components. Full flux at resting potential.

Core

fermi_energy_sim.py

Fermi energy for Na, Cu, Al, Au, Ag, Fe. Temperature dependence. Fermi temperature vs body temp. Wearable contact conductivity.

Core

lattice_energy_sim.py

Born-Lande for NaCl, CsCl, ZnS, MgO, CaF2, TiO2. Energy vs spacing curves. Stability ranking. Biocompatible material selection.

Core

thermoelectric_model.py

ZT figure of merit for Bi2Te3, PbTe, SnSe, Organic. Power output at body temperature. Reality check vs BLE/MCU/phone power needs.
